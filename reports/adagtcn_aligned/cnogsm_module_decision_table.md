## Short Decision Summary

| Decision | Modules |
|---|---|
| Keep as core candidates | Word-level sequence extraction; train-only subject-split dataset loader; EEG-only graph SSM; gaze-only SSM; gaze-controlled state-space transition; full CNO-GSM stack |
| Supplementary until proven | Mask handling; EEG adaptive graph encoder; TCN baseline; neuro-oculomotor bipartite graph; subject-invariant bridge; common-unique disentanglement; effect audit script |
| Drop from main claims for now | Any claim that CNO-GSM exceeds original AdaGTCN; any claim of raw EEG waveform modeling; any claim that bipartite graph, subject bridge, or common-unique disentanglement is independently effective |

## Main Contribution Candidate After Filtering

The safest main contribution is:

> Under an AdaGTCN-aligned available-Y subject-independent protocol, word-level EEG band representations and eye-tracking sequences can be modeled as reading-state dynamics with state-space temporal encoders. Eye-tracking sequence dynamics are a strong baseline, EEG graph-state representations add complementary evidence, and the integrated CNO-GSM stack is promising but requires leave-one-module-out validation before all submodules can be claimed.

## Required Next Evidence Before Writing Main Claims

1. Fix or replace the `adagtcn_aligned` TCN baseline because its full-data history reports `loss=nan`.
2. Run 3-5 seeds for `adagtcn_aligned`, `eeg_only_graph_tcn`, `eeg_graph_ssm`, `gaze_only_ssm`, `gaze_control_ssm`, and `full_cnogsm`.
3. Add full-data isolated ablations for `bipartite_graph_ssm`, `bridge_bipartite_ssm`, and `full_no_common_unique`.
4. Add no-graph EEG SSM and aggregate-gaze nonsequence baselines.
5. Run label-shuffle, modality permutation, and missingness permutation sanity checks.
