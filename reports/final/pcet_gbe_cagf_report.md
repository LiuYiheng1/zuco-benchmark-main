# PCET + GBE + CAGF Experiment Report

## Overview

This report presents results for the new PCET + GBE + CAGF model architecture.

## Key Changes from GETA version

- **GBE (Gaze Behavioral Encoding)** directly uses gaze features
- No entropy/confidence calculation
- No gaze attention reweighting of EEG
- CAGF fusion remains the same

## Experimental Protocol

- Few-shot personalized calibration
- LOSO target subject
- k = 3, 5, 10, 20, 50 shots per class
- 16 Y-subjects
- 5 seeds
- 50% calibration / 50% test split
- No test leakage
